# Tinstop Website Intelligence API

A keyless, machine-payable Website Intelligence API for domain security and performance audits, built for AI agents. Access is gated per-request via the x402 payment protocol using USDC on Base — no accounts or API keys. Offers 16 pay-per-call endpoints covering DNS/SSL/TLS checks, email authentication (SPF/DKIM/DMARC), HTTP security headers, web performance (PageSpeed), domain recon, and AI-generated audit reports.
